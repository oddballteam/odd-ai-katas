## The way we write code is changing

The model behind your tool matters less than the interface you use to work with it. Here's what the landscape looks like right now.

## Terminal agents

Lightweight, portable, and easy to run multiple sessions at once. No IDE overhead in the context window.

**Examples**: [Claude Code](https://claude.ai/download), [Codex CLI](https://github.com/openai/codex), [Gemini CLI](https://github.com/google-gemini/gemini-cli), [GitHub Copilot CLI](https://docs.github.com/en/copilot/github-copilot-in-the-cli)

- **[Claude Code](https://claude.ai/download)** popularized the terminal agent category. Has the most built-in features of any CLI agent. Not open source, but the source was [accidentally published](https://findskill.ai/blog/is-claude-code-open-source/) in March 2026. No data retention for training.
- **[Codex CLI](https://github.com/openai/codex)** is open source (Rust). Telemetry is opt-in and disabled by default.
- **[Gemini CLI](https://github.com/google-gemini/gemini-cli)** is open source. Telemetry configurable via `.gemini/settings.json`.
- **[GitHub Copilot CLI](https://docs.github.com/en/copilot/github-copilot-in-the-cli)** is closed source. Telemetry opt-in. Supports `COPILOT_OFFLINE=true` to disable all telemetry.

## IDEs

Completions, chat, agent modes. Your code stays front and center. This is mostly about preference.

**Examples**: [VS Code + Copilot](https://code.visualstudio.com), [Cursor](https://cursor.com), [Antigravity](https://antigravity.dev), [Windsurf](https://windsurf.com)

- **[VS Code + Copilot](https://code.visualstudio.com)** is the biggest ecosystem. VS Code is open source, Copilot is not. Telemetry enabled by default but configurable. Enterprise tier offers no-code-retention.
- **[Cursor](https://cursor.com)** is closed source. Cursor 3 (April 2026) added an Agents Window for running parallel agents across repos. Privacy Mode disables telemetry and code retention.
- **[Antigravity](https://antigravity.dev)** is closed source (Google, Gemini-powered). Has both an Editor view and a Manager view for orchestrating multiple agents in parallel.
- **[Windsurf](https://windsurf.com)** is a VS Code fork (open source base, closed extensions). Cascade streams agent changes in real time with automatic context retrieval.

## Agent-first workflows

Every major tool is building a dedicated surface for dispatching and managing multiple agents. Less editing, more steering.

- **[Claude desktop app](https://claude.ai)** visual diffs, drag-and-drop pane layout, parallel sessions with git isolation. Good entry point if you don't live in the terminal.
- **[Codex App](https://openai.com/codex/)** is a standalone command center. Parallel agents in isolated worktrees. Free tier available.
- **[Cursor Agents Window](https://cursor.com)** (Cursor 3, April 2026) is a full-screen tiled workspace for running agents across repos, locally or in the cloud.
- **[VS Code Agents app](https://code.visualstudio.com/blogs/2026/02/05/multi-agent-development)** is a companion app for running agents across multiple repos in parallel. Expected May 2026.

## The same agent, everywhere

The biggest shift in 2026: the tool you use matters less than the agent behind it. The same agent now shows up across terminal, IDE, desktop, web, and phone. Pick the one you trust, then use it from whatever surface fits the moment.

GitHub's Copilot CLI also supports [remote control](https://github.blog/changelog/2026-04-13-remote-control-cli-sessions-on-web-and-mobile-in-public-preview/) -- start a session in your terminal and steer it from your phone.

## Voice typing

You're typing prompts all day. Speaking is 150+ WPM, typing is 50-80 for most developers. Voice typing is a natural fit.

- [Wispr Flow](https://wisprflow.ai). Integrates with Cursor, VS Code, Windsurf. Cross-platform. $12/mo.
- [Voicy](https://usevoicy.com). Works in every app. 99%+ accuracy. Cross-platform.
- [SuperWhisper](https://superwhisper.com). On-device, privacy-focused. Mac only.
- **Built-in dictation**. macOS and Windows. Free. Good enough to start.

Cursor and Antigravity also have mic buttons built into the chat input.

## The lines are blurring

Most tools span multiple categories now:
- Cursor has an IDE and an agent window
- Claude Code runs interactively or as a background agent
- GitHub Copilot works in the CLI, the IDE, and the browser

Most developers use 2-3 tools. The question isn't "which is best" but "which interface fits what you're doing right now."
