The tools have mostly converged on what is possible. What differs is naming. This kata is a reverse index: pick the task, look up the command in your tool.

For each tool, below is the full built-in surface as of April 2026. Start by skimming by task, then scan the "everything else" section to see what you were not even looking for.

## What's new in the last two months

<Tabs labels={["Claude Code", "Cursor", "Copilot (VS Code)", "Antigravity"]}>

<div class="kata-tab-panel">

- `/ultraplan <prompt>` and `/ultrareview [PR]` - multi-agent plan/review in the cloud.
- `/batch <instruction>` - spread work across 5-30 worktrees.
- `/autofix-pr [prompt]` - watch a PR and fix CI failures.
- `/loop [interval]` (alias `/proactive`) and `/schedule` (alias `/routines`) - run prompts on a timer or cron.
- `/teleport` (`/tp`) - pull a web session into the terminal.
- `/rewind` (aliases `/checkpoint`, `/undo`) and `/branch` (alias `/fork`) - time-travel conversation and code.
- `/btw <question>` - aside that does not touch the main thread.
- `/insights`, `/recap`, `/team-onboarding`, `/powerup`, `/voice`, `/fast`, `/tui`.
- MCP prompts auto-appear as `/mcp__<server>__<prompt>`.

</div>

<div class="kata-tab-panel">

- `/plan` and `/ask` modes - planning-first and read-only modes replace the old action slashes.
- `/auto-run on|off|status` - explicit control over autonomous execution.
- `/max-mode` - long-context mode on supporting models.
- `/sandbox` - scope FS and network.
- `/compress` - in-place conversation summary.
- `/create-skill` and `/migrate-to-skills` (2.4+) - generate reusable skills and convert older rules/commands.
- `/mcp list|enable|disable` - toggle MCP servers without leaving chat.
- Old action slashes (`/edit`, `/fix`, `/explain`, `/test`, `/docs`, `/commit`) removed; the agent decides.
- Context `@`s (`@Web`, `@Git`, `@Definitions`, `@Linter Errors`, `@Codebase`, `@Terminal`) removed; agent gathers automatically.

</div>

<div class="kata-tab-panel">

- `/plan` in Agent mode.
- `/troubleshoot` - analyze agent debug logs.
- `/yolo` (alias `/autoApprove`) and `/disableYolo` - Agent auto-approval toggles.
- AI-authored customizations: `/create-agent`, `/create-skill`, `/create-prompt`, `/create-instruction`, `/create-hook`.
- Config UIs: `/agents`, `/hooks`, `/instructions`, `/prompts`, `/skills`.
- `/init` in Agent generates AGENTS.md / copilot-instructions.
- `/fork` splits a session; `/compact` summarizes in place.
- `@workspace` retired - workspace context is implicit in Ask/Edit/Agent.

</div>

<div class="kata-tab-panel">

- Planning Mode in the Manager selector - agent produces a Plan artifact before coding.

- Browser subagent - agent self-drives a browser for verification.
- Artifacts panel (Trust Layer): Plan, Task List, Walkthrough, Screenshots, Browser Recordings.
- `@<MCP-server>` for each configured MCP (Supabase, Firebase, Linear, Vercel, Cloudflare).

## Attach files and keyboard shortcuts

<Tabs labels={["Claude Code", "Cursor", "Copilot (VS Code)", "Antigravity"]}>

<div class="kata-tab-panel">

**Attach files**
- `@<path>` in the prompt fuzzy-matches files and inserts them as context.
- Drag-drop files or images into the terminal (supported terminals); paste images directly (iTerm2, Ghostty, Alacritty).
- `/add-dir <path>` extends working roots for the session.
- `/memory` edits CLAUDE.md, the always-loaded instructions file.

**Keyboard shortcuts**
- `Shift+Tab` cycle plan / accept / auto-accept modes.
- `Esc` interrupt current turn; double-`Esc` opens rewind picker.
- `Ctrl+R` toggle verbose output; `Ctrl+L` clear screen (context intact).
- `Shift+Enter` newline (after `/terminal-setup`); `Enter` submits.
- `Tab` completes `/command`, `@path`, `#memory`.
- `Up` cycles prior prompts.

</div>

<div class="kata-tab-panel">

**Attach files**
- `@Files`, `@Folders`, `@Code` (symbols), `@Docs`, `@Past Chats`.
- Drag-drop files and images into the chat input.
- Paste images from the clipboard.
- `.cursorignore` controls what gets indexed.

**Keyboard shortcuts**
- `Cmd/Ctrl+K` inline edit on selection; no selection = generate at cursor.
- `Cmd/Ctrl+Shift+Enter` inside Cmd-K applies file-wide.
- `Alt+Enter` inside Cmd-K asks a question; reply "do it" to convert.
- `Cmd/Ctrl+L` send selection to Chat / toggle Chat pane.
- `Cmd/Ctrl+I` open Composer / Agent panel.
- `Cmd/Ctrl+Shift+L` new chat.
- Terminal `Cmd/Ctrl+K` natural language to shell command.
- `Tab` accept inline completion.

</div>

<div class="kata-tab-panel">

**Attach files**
- `#file:<name>` attaches a file. Other typed refs: `#folder:`, `#selection`, `#problems`, `#codebase`, `#testFailure`, `#terminalLastCommand`, `#terminalSelection`.
- Drag from Explorer into Chat.
- Paste images from clipboard; paperclip icon for uploads.
- `.github/copilot-instructions.md` and `AGENTS.md` auto-load.

**Keyboard shortcuts**
- `Cmd/Ctrl+I` Inline Chat on selection.
- `Cmd/Ctrl+Alt+I` toggle Chat view.
- `Cmd/Ctrl+Shift+I` open Quick Chat.
- `Cmd/Ctrl+L` clear / new chat.
- `Cmd/Ctrl+Enter` accept Inline; `Esc` dismiss.
- `Cmd/Ctrl+Right` accept next word of a completion.
- `Cmd/Ctrl+Shift+A` open next edit suggestion.

</div>

<div class="kata-tab-panel">

**Attach files**
- `@file`, `@folder`, `@codebase`, `@workspace`, `@selection`, `@terminal`, `@problems`.
- `@url` / `@web` fetch URLs; `@docs` references indexed docs.
- Drag-drop files and images into the Agent panel.
- `.agent/workflows/<name>.md` are your slash commands; `AGENTS.md` auto-loads.

**Keyboard shortcuts**
- `Cmd/Ctrl+I` inline edit / natural-language terminal command.
- `Cmd/Ctrl+L` toggle Agent panel.
- `Cmd/Ctrl+Shift+I` open Agent panel.
- `Cmd/Ctrl+Shift+L` new conversation.
- `Tab` accept inline completion.

</div>

</Tabs>

## Exercise

1. Pick a row from the table for your tool that you did not know about. Run it.
2. Then pick the same row for a tool you do not use. Compare: does your tool have an equivalent? If not, is it a gap or a philosophy choice?
3. Ask the AI: "given what I just did, which built-in would have replaced the last 5 prompts I sent?"
