## Attach files and keyboard shortcuts

<Tabs labels={["Claude Code", "Cursor", "Copilot (VS Code)", "Antigravity"]}>

<div class="kata-tab-panel">

**Attach files**
- `@<path>` in the prompt fuzzy-matches files and inserts them as context.
- Drag-drop files or images into the terminal (supported terminals); paste images directly (iTerm2, Ghostty, Alacritty).
- `/add-dir <path>` extends working roots for the session.
- `/memory` edits CLAUDE.md, the always-loaded instructions file.

**Keyboard shortcuts**
- `Shift+Tab` cycle plan / accept / auto-accept modes.
- `Esc` interrupt current turn; double-`Esc` opens rewind picker.
- `Ctrl+R` toggle verbose output; `Ctrl+L` clear screen (context intact).
- `Shift+Enter` newline (after `/terminal-setup`); `Enter` submits.
- `Tab` completes `/command`, `@path`, `#memory`.
- `Up` cycles prior prompts.

</div>

### Where each surface lives

Claude Code spans four surfaces. Some features only exist on some of them.

**Terminal CLI only**
- `Monitor` tool - tails logs, test runners, deploys, PR checks into the conversation.
- Fullscreen TUI (`/tui`), color themes, statusline, keybindings.

**Desktop app only** (launch with `/desktop` or `/app`)
- **Drag-and-drop pane layout** - chat, diff, preview, terminal, file, tasks panes.
- **Integrated file editor** - spot edits with save/discard, not just patches through the agent.
- **Visual diff review with inline comments** - line-by-line feedback that Claude revises against.
- **Live app preview pane** - embedded browser for dev servers, HTML, PDFs with auto-verification.
- **Computer use** - Claude drives your screen: clicks, types, automates GUI tasks (macOS/Windows).
- **Dispatch** - kick off tasks from your phone into the Desktop app.
- **Native connectors UI** - visual setup for Slack, GitHub, Linear, Google Calendar.
- **View modes (Verbose / Normal / Summary)** - collapse transcripts or see every tool call.
- **GitHub PR status bar** - monitor PRs, auto-fix CI failures, auto-merge when green.

**Web (claude.ai/code)**
- **Routines** - cloud-scheduled or webhook-triggered Claude runs. Laptop stays off. Managed in the web UI; `/schedule` in the CLI talks to the same backend. Pro 5/day, Max 15/day, Team/Enterprise 25/day.
- **Cloud sessions** - long-running jobs that persist when you close any client. Teleport into the terminal with `/teleport`.

**Cross-surface**
- Parallel sessions with worktree isolation (Desktop + web).
- Remote control (`/remote-control`) - pair claude.ai with any terminal session.
- Mobile app - read notifications, kick off Dispatch tasks, continue on desktop.

<div class="callout-label">Different model</div>
Antigravity does not ship named built-in slashes. Typing `/` lists workflow files under `.agent/workflows/` - and nothing else. If `/` is blank, it is because you have no workflow files, or the YAML frontmatter is malformed (must include `description:`), or you have hit the known parser crash that shows "0 results". The `…` (more) dropdown in the Agent panel is the same list and often works when `/` does not.

The table below shows what is actually native: @ mentions, keyboard actions, and Manager features.

## Exercise

Try one new command or feature you have not used before. Suggestions:

- **Claude Code**: `/insights` for a report on how you have been using it this session, or `/stats` for your daily usage, streaks, and model preferences.
- **Cursor, VS Code + Copilot, Antigravity**: the chat window has icons around the input box (paperclip, model picker, mode switcher, microphone, etc.). Hover each one to read the tooltip.
